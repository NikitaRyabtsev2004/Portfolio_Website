Для использования в образовательном контексте:

## Использование

1. Клонируйте репозиторий:

git clone https://github.com/NikitaRyabtsev2004/Portfolio_Website

2. Перейдите в директорию проекта:

cd bit-blitz

3. Установите зависимости:

npm install

4. Запуск клиента:

npm run start

5. Запуск сервера:

cd server
nodemon server.js
